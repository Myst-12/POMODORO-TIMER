# Pomodoro Timer

A simple, interactive Pomodoro timer built with HTML, CSS, and Vanilla JavaScript.

https://github.com/user-attachments/assets/b99748f8-fad3-4588-a263-ce5cb4150091

## Features

- **Work Mode:** Choose between 25 or 50-minute focus sessions.
- **Break Mode:** Choose between 5 or 10-minute breaks with automatic visual transitions.
- **Controls:** Start, Stop, and Reset the timer with ease.

## How to Run

Simply open the `index.html` file in any modern web browser to start using the timer.
